Library Management System

The Library Management System is a full-stack web application built using modern web technologies:

  • Search for books in the database.
  • Borrow and return books through an interactive UI.
  • Manage book inventory via an admin panel.
  • Secure user authentication using Firebase.

Programming Languages

  • JavaScript, SQL

Applications and Tools

  • React.js, Node.js, Firebase, PostgreSQL
