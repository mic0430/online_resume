Spark-Art

Spark-Art is an online marketplace that allows artists to share, showcase, and sell their artwork. The platform features:

  • Interactive UI for artists to upload and manage artwork.
  • Secure payment processing for digital art transactions.
  • AI-based recommendations for trending artworks.
  • User authentication and social engagement features.

Programming Languages

  • JavaScript, Python

Applications and Tools

  • Next.js, Python (Flask), Firebase, Azure
